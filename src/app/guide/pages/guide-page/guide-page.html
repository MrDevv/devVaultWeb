DevVault API Documentación

Introducción

Una vez que tienes tu perfil cargado en DevVault, el siguiente paso es conectarlo a tu portafolio. Para eso existe esta API: un conjunto de 4 endpoints que puedes consultar desde cualquier frontend usando tu API KEY personal, que generas luego de registrarte.

Esta guía cubre todo lo que necesitas para hacer tus primeras peticiones: cómo incluir tu KEY en los headers, qué devuelve cada endpoint y qué esperar en la respuesta.

Autenticación

Todas las peticiones requieren enviar la API Key en el header:

Authorization: Bearer TU_API_KEY

Ejemplo:

Authorization: Bearer 691c3e42-9f1d-162e-9126-30c4d524abca-2032023165314

Base URL

https://servermrdevv2.duckdns.org/portfolio/api/v1

Endpoints

1. Obtener datos del desarrollador

Endpoint

GET /desarrolladores/me/datos

Descripción

Obtiene la información general del desarrollador:

  • desarrollador ID
  • nombres
  • apellidos
  • correo
  • github url
  • linkedin url
  • cv url
  • logo url
  • prefijo telefónico
  • teléfono
  • biografía
  • puesto

Ejemplo respuesta JSON

{{ data_desarrollador | json }}
